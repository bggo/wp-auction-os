== Description ==

Central Wordpress theme is simply solution to customize your website. It`s clean, simply and very powerful tool for organizing your blog or others categories of websites..
You may change some design elements from Admin UI, like: background of page, header text-color, header background-image.

== Installation ==

Manual installation:

1. Upload the `central` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress

Installation using "Add New Theme":

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'newlife'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button
5. Activate the theme through the 'Themes' menu in WordPress

== Theme Notes ==

= SLider =

For slider you need to do 3 steps:
1. You need to create a new category or choose from existing categories of posts which you want to display in slider section on home page.
2. If category not have posts yet you need create some posts. In slider will by displaying a title of post, content and featured image of post ).
   Slider displaying only posts with "draft" status so you need set this status for every post which you want to see in slider. 
3. You need go to "Apperance"->"Slider" menu item in Admin UI and choose category which you want to displaying as slider.
( for featured images of slider we recommend you using the proportions 900x218px (width/height) ).

= Featured Images =

For featured images of posts or pages we recommend using the proportions 620x195px (width/height).

= Widgets =

The Theme includes a standart widgets.
By default in sidebar will display widgets: search form, recent posts, recent comments, archive and categories.
You can change the set of widgets on your own.
